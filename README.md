# Niriha: A Minimal, Fast, and Serene Wayland Setup

Niriha is a lightweight and efficient Wayland setup featuring **Niri WM**, **Waybar**, **Walker launcher**, and other essential tools for a sleek, fast, and customizable user experience. Ideal for users who prioritize performance and simplicity on Wayland.

## Features

- **Niri WM**: Lightweight and fast Wayland compositor.
- **Waybar**: Fully customizable top bar with system info.
- **Walker**: A fast app launcher for seamless workflow.
- **Mako**: Notification daemon with a simple design.
- **Wallpaper & Display Management**: `waypaper` and `nwg-displays` for display configuration.
- **System Monitoring**: `btop++` to monitor system resources.
- **Theming**: Customize your appearance with `nwg-look`.
- **Alacritty**: A fast, GPU-accelerated terminal emulator.
- **Kitty**: Another fast, feature-rich, GPU-based terminal emulator.
- **Fastfetch**: A modern command-line system information tool.
