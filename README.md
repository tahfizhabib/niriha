## Features

- Niri WM: Lightweight Wayland compositor  
- Waybar: Fully customizable top bar  
- Walker: App launcher  
- Mako: Notification daemon  
- Wallpaper & Display Management: waypaper, nwg-displays  
- System Monitoring: btop++  
- Themes & Appearance: nwg-look
